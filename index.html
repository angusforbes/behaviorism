behaviorism

A graphics and algorithms framework based on JOGL bindings

Behaviorism is an 3D information visualization framework. It is built using the Java for OpenGL bindings. It includes the following features: - a flexible scene-graph architecture which mimimizes state changes and modelview transformations. - extensible library for common 2D and 3D textured geometric shapes and surfaces including curves, spheres, convex and non-convex polygons, and wrappers for GLU quadrics, GLUT primitives, and NURBS. - a timing/event framework for attaching behaviors to geometry, cameras, or to other behaviors or sequences of behaviors. - wrapper for the java media components to play and control video files in various formats. - level-of-detail algortihm for nice looking text in 3D. - library of utilities for matrix transformations, geometric algorithms, combination effects, color management, and timing. - input handlers for mouse, keyboard, wii, 3d mouse. - web services library for youtube, yahoo, google, flickr. - a set of layout algorithms, including self-organizing maps and a bin-packing implementation.

Dependencies

Java 6+, JOGL 1.1+ native libraries, MySQL 5.0+, Java Media Components (via JavaFX 1.0).

Install

The repository contains an entire Netbeans project. You will need to put all .jar files in the libs directory on your classpath. You will also need to link to native libraries for your os at runtime. The natives are stored in the natives folder. You can link to the appropriate native libraries using the -Djava.library.path option.
